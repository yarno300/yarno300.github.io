Code Breaker Engine

Code Breaker is a game engine for creating graphical applications using the Pygame library and the programming language Python. It provides a simple coding interface for creating windows, rendering graphics, and handling user input. It is and engine to create games fast and easy!

Docs Github

Why Code Breaker?

Here are a few reasons why you should choose for Code Breaker Engine
Ease of use

We focused to make our engine as easy as possible, so you can make your product fast and hassle free. Cannot find a feature? Just use one of Pygame!

Fast

We are leveraging pythons speed and easiness and the power of Pygame together to get the best result. This makes out engine easy but still fast.

Open-Source

Our engine is open-source, so we are welcoming you to help us! Found a bug? Then try to fix it or create a issue!
